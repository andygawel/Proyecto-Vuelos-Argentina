# ✈️ Argentina Air Traffic Analysis

## 📌 Overview

This project analyzes air traffic in Argentina using Power BI, SQL, and a normalized relational data model.

The goal is to transform raw flight data into interactive dashboards that help answer business questions about airport activity, passenger traffic, and flight operations.

![Dashboard Overview](images/Portada.png)

## 🛠️ Tools

- Power BI
- SQL
- DAX
- Power Query
- Data Modeling
- Data Visualization

## 🎯 Business Questions

This project was developed to answer key business questions, including:

- Which airports have the highest flight traffic?
- Which cities concentrate the largest number of passengers?
- What are the busiest flight routes?
- How are domestic and international flights distributed?
- Which airports show the highest operational activity?

## 📈 Key Insights

- Air traffic is highly concentrated in Buenos Aires, which records approximately 900% more flights than the following provinces.
- Around 80% of all flights are domestic, highlighting the importance of Argentina's internal air transportation network.
- International operations are concentrated in a relatively small group of destination countries, indicating that international connectivity depends on a limited number of routes.
- Some unexpected patterns, such as the high number of flights associated with Cabo Verde, suggest opportunities for further investigation and demonstrate the importance of validating unusual findings before drawing business conclusions.
